## 🧠 Overview

**Data-Butler-Pro** is an AI-powered interactive data analytics dashboard built with Streamlit.  
It enables users to upload datasets (CSV/Excel), automatically clean data, perform exploratory data analysis (EDA), visualize insights, and generate business intelligence reports.

This project simulates a **self-service Business Intelligence (BI) tool**, similar in concept to Power BI, but built entirely using Python.

---

## 🎯 Key Features

### 📂 Data Handling
- Upload CSV / Excel files
- Automatic dataset preview
- Missing value detection

### 🧹 Data Cleaning
- Handle missing values
- Basic preprocessing
- Data type detection

### 📊 Exploratory Data Analysis (EDA)
- Summary statistics
- Column-wise analysis
- Correlation analysis
- Distribution insights

### 📈 Data Visualization
- Interactive charts using Plotly
- Bar charts, line charts, histograms
- Correlation heatmaps

### 🧠 Business Insights
- Automated insights generation
- Trend detection
- Data pattern analysis

### 📄 Reporting
- Structured analysis output
- Export-ready insights (dashboard-style reporting)

## 💡 Project Use Case

This tool is designed for:

- Data Analysts
- Business Intelligence Developers
- Students learning data analysis
- Small businesses needing quick insights

It acts as a **lightweight BI alternative** for fast data exploration without complex tools.
